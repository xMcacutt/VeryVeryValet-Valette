# Very Very Valet - Valette

## What Does This Do?
***Requires Controller to Change Valet Colours***

This mod drastically overhauls the valet colour selection system.
Instead of having predefined skin colour sets, you can now select virtually any colour.

To enter skin edit mode, press the button that is normally used to cycle the current skin.
Once in skin mode, moving the analog stick will change the colour of the currently selected element.
- Moving left and right will cycle the hue.
- Moving up and down will cycle the saturation/lightness.
- Whether you are currently editing the saturation or lightness can be toggled between by pressing the right shoulder button.
- The currently selected element can be cycled by pressing the left shoulder button. 

Each character has three elements except `Comms`.

## Installation
This mod can be installed through the [ThunderStore Mod Manager](https://thunderstore.io/c/very-very-valet/)

## Manual Install Guide
1. Download [BepinEx](https://github.com/BepInEx/BepInEx/releases) 5.4 for x64 architecture
2. Extract all files into your `Very Very Valet/windows` directory
3. Launch the game
4. Close the game
5. Go to BepinEx/config and open BepinEx.cfg in a text editor
6. Change HideManagerGameObject to true
7. Download the [latest release of this mod](https://github.com/xMcacutt-Archipelago/VeryVeryValet-APClient/releases)
8. Extract the Parkipelago folder into the BepinEx/plugins folder in your game folder
9. Launch the game
10. Pick up cars
11. Park them
12. Cry
